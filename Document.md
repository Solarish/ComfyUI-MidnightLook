# ComfyUI-MidnightLook: คู่มือการใช้งาน Custom Nodes

ชุด Custom Nodes ของ `ComfyUI-MidnightLook` ถูกพัฒนาขึ้นเพื่อเสริมประสิทธิภาพการทำงานเฉพาะทางใน ComfyUI โดยเน้นที่การทำ Image detailing, Upscaling, การจัดการ Workflow สมัยใหม่แบบ Loop และเทคโนโลยีวิเคราะห์สัดส่วนภาพ

ด้านล่างคือ **แนวคิด (Concept)** และ **วิธีใช้งาน (Usage)** ของ Node หลักๆ ในแพ็กเกจนี้

---

## 1. กลุ่ม Midnight Detailer (การทำมาสก์และรีทัชเฉพาะจุด)

### 1.1 SAM2 Loader (ML)
**แนวคิด (Concept):** 
รวมพลังระหว่าง **GroundingDINO** (AI ตามหาสิ่งของจากคีย์เวิร์ด) และ **SAM2** (AI ตีเส้นกรอบ/Mask สุดแม่นยำจาก Meta) เพื่อสร้างหน้ากาก (Mask) อัตโนมัติโดยที่ผู้ใช้ไม่ต้องมาระบายสีเอง (Zero-shot segmentation)
**วิธีใช้:**
- นำรูปภาพที่ต้องการลบ/แก้ มาเสียบเข้าที่ช่อง `image`
- พิมพ์คีย์เวิร์ดสิ่งที่ต้องการรีทัชในช่อง `prompt` (เช่น `face`, `eyes`, `dog`)
- `box_threshold` และ `text_threshold` ปรับตามความเหมาะสม (ค่าเริ่มต้นแนะนำที่ 0.3 และ 0.25 ตามลำดับ หากหาของยาก ปรับลดเพื่อคลายสเปกการค้นหา)
**สิ่งที่ได้ (Output):** 
BBOX (ตำแหน่งพิกัดกรอบสี่เหลี่ยม) และ MASK (หน้ากากขาวดำพร้อมนำไปใช้งาน)

### 1.2 Midnight Detailer (ML)
**แนวคิด (Concept):** 
ตัวสร้างรายละเอียด! รับไม้ต่อจากหน้ากาก (Mask) โดยจะทำหน้าที่ **"ครอบตัดส่วนนั้น (Crop) -> ขยายใหญ่แล้ววาดใหม่ (Inpaint) -> แปะคืนให้เนียน (Stitch)"** เฉพาะพื้นที่เป้าหมายโดยอัตโนมัติ ช่วยประหยัดเวลา Render เพราะเราเจาะแก้เฉพาะจุด และได้รายละเอียดระดับ Micro-detail
**วิธีใช้:**
- ต้องป้อน `image`, `mask` และข้อมูลโมเดลทั้งหมดที่จำเป็นต่อการวาดรูป (Model, VAE, Positive, Negative) ให้ครบถ้วนในจุดเดียว
- เสียบเข้ากับพารามิเตอร์ของ KSampler ปกติ (Steps, CFG, Sampler, Denoise) 
- Node นี้จะทำงานอยู่เบื้องหลังเงียบๆ และส่งไพพ์รูปที่เสร็จสมบูรณ์ออกมาทางท่อ Output ทันที

## 2. กลุ่ม Iterative Upscaling (ขยายภาพความละเอียดสูงแบบแบ่งช่วง)

### 2.1 Sample Upscaler Provider (ML)
**แนวคิด (Concept):**
เปรียบเสมือน "กระเป๋าเครื่องมือ" ที่มัดรวมของหนักๆ KSampler, VAE, โมเดลแว่นขยาย (Upscale Model) และเงื่อนไข Prompt เข้าไว้ด้วยกัน (Type: `UPSCALER_PROVIDER`) เพื่อแก้ปัญหาสายระโยงระยาง (Spaghetti Nodes) ใน Workflow ที่มีความซับซ้อนสูง
**วิธีใช้:**
- ต่อสายโมเดลทุกอย่างเข้ากับตัวนี้ แล้วลากสาย Output ที่ชื่อ `upscaler` สายเดียวไปส่งต่อให้ Node ถัดไป (ง่ายและสะอาดสมองมาก)

### 2.2 Midnight Upscaler (Iterative)
**แนวคิด (Concept):**
ปัญหาส่วนใหญ่ของการสั่งขยายภาพทีเดียว 2X คือภาพมักจะสูญเสียโครงสร้างหรือรายละเอียดเปลี่ยนไป Node นี้จะใช้วิธีเรนเดอร์ภาพแบบ **"ทยอยขยายหน้าจอทีละนิด (Iterative)"** หั่นการทำงานเป็นแบ่งซอยย่อย 3 ครั้งต่อการเคลียร์ภาพ เพื่อให้สเกลภาพยังคงเค้าโครงเดิม และค่อยๆ เติมรายละเอียด (Denoise) อย่างเป็นธรรมชาติ
**วิธีใช้:**
- รับกระเป๋าเครื่องมือจาก `upscaler` แล้วเสียบรูปที่ต้องการขยายเข้าที่ `image`
- กำหนดค่า `scale_factor` ว่าอยากขยายรวมกี่เท่า และตั้งรอบทำซ้ำ (`iterations`) ซึ่งค่ามาตรฐานคือ 3 รอบ
- กะเกณฑ์ความเพี้ยนด้วย `denoise_start` กับ `denoise_edge`

## 3. กลุ่ม Loop Control (การทำซ้ำเงื่อนไขแบบเป็นวงกลม)

**แนวคิด (Concept):**
ทะลวงข้อจำกัดของ ComfyUI ที่แต่เดิมต้องประมวลผลเป็นเส้นตรงรวดเดียวจบ (DAG) ให้สามารถทำงานเป็นวงจร "ลูปวน (Loop)" หักเหทิศทางได้แบบอัลกอริธึม (เช็คจนถูกใจถึงยอมให้ผ่านด่าน)

### 3.1 Loop Start
- ใช้เป็นจุดเริ่มต้นเชือก! ทำหน้าที่ตรวจสอบว่าการวิ่งครั้งนี้เป็นการวิ่งรอบใหม่ (Iteration) หรือรอบแรกสุด 
- ส่งสายไปจุดระบุเงื่อนไขต่างๆ 

### 3.2 Loop End / Condition
- จุดหยุดตรวจพาสปอร์ต! Node รับค่า `should_continue` (มักจะเป็นการเซ็นเซอร์รูปภาพ หรือตรวจเช็ค Faces แบบ Auto)
- ถ้าระบบโยนผลมาว่า `True` ตัว Loop End จะแอบสั่ง ComfyUI API เพื่อจับ Queue รันโปรเจกต์นี้ซ้ำใหม่ตั้งแต่ต้นทันที (Re-queueing) โดยดัดแปลงรหัส Seed ป้องกันไม่ให้วาดของเก่าออกมาจนกว่าจะผ่านด่านเป็น `False`

## 4. กลุ่ม Image Crop/Analysis (วิเคราะห์และเจาะจงใบหน้า)

### 4.1 DeepFace FaceCrop / Mediapipe
**แนวคิด (Concept):** 
ใช้อัลกอริธึมคณิตศาสตร์วิชันในการสแกนสัดส่วนโครงสร้างใบหน้า (Facial Landmarks) ที่หาดวงตา จมูก ปาก เหนือกว่าการตรวจจับทั่วไป เหมาะสำหรับงานโฟกัสทำ FaceID / DeepFake Pipeline และ LoRA Dataset
**วิธีใช้:**
- ใส่ป้ายชื่อรูปและระบุ `margin` (ขอบที่เอาเผื่อ)
- หากระบุ `target_size` รูปขอบหน้าจะถูกยืดหดให้อยู่ในกรอบประมวลผลเป๊ะๆ ตามที่เราอยากให้มันเป็น

---

## 5. กลุ่ม VLM Prompter (ระบบอ่านภาพพรรณนา)

### Qwen2.5-VL Image Prompter
**แนวคิด (Concept):**
จับ AI สายตาอัจฉริยะแบบฝังตัวเครื่อง (Local Vision-Language Model) มาอ่านรูปภาพโดยไม่ต้องส่งรูปไปขึ้น Cloud 
**วิธีใช้:**
- ส่งภาพให้มันและพิมพ์ถามคำถาม เช่น "ช่วยแต่งตัวละครคนนี้ให้กลายเป็น Cyberpunk หน่อย ในสไตล์เดิม" ทิ้งไว้ช่อง Prompt
- ดึง String อักขระออกไปร้อยเรียงแต่งภาพ Inpainting ต่อได้อย่างสร้างสรรค์
